# Admin_Fstep
What Linux Admin done for First time


# Download

1) First You shoul be root:
sudo su -
 
2) Get File
wget https://github.com/kooshakoosha/Admin_Fstep/archive/refs/heads/main.zip

3) unzip File
unzip main.zip

4) Go to Directory
cd Admin_Fstep-main

# RUN

Note: You should be root to Run script 

./Fstep
